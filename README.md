# MindAlive — Frontend Android

Aplicación Android para acompañar a personas mayores, mantener 
su mente activa y facilitar la supervisión por parte del cuidador.

## ¿Qué hace?

Permite al mayor hablar con Mindi, un asistente de voz con IA, 
y realizar ejercicios cognitivos adaptados a su nivel. El cuidador 
dispone de un panel para gestionar alarmas, el perfil del mayor 
y ver la evolución en gráficas.

## Stack tecnológico

- **Lenguaje** — Kotlin
- **Mínimo SDK** — API 26 (Android 8.0)
- **Voz** — SpeechRecognizer + TextToSpeech
- **Red** — OkHttp
- **Notificaciones** — WorkManager

## Pantallas principales

- Login y registro
- Pantalla del mayor: HABLAR y JUGAR
- Cuatro juegos cognitivos: Simón, Memoria, Operaciones, El Raro
- Panel del cuidador: alarmas, perfil, centro de control

## Requisitos para ejecutar

- Android Studio
- Emulador o dispositivo con Android 8.0 o superior
- Backend corriendo en local o en red
